Download Obscura

One app: a non-custodial wallet, the cross-chain swap desk, and one-click mining — running on your own node, so your keys and your activity never leave your machine. Single static binary, no dependencies.

The app runs your own node, so (unlike the hosted web wallet) your IP and balance scans stay private to you. Back up your recovery phrase: it is stored only on this machine.

Run it

  1. Unzip the download.
  2. macOS: open the Obscura.app. Windows: double-click Obscura.bat. Linux: run ./obscura.
  3. A native app window opens with your wallet, swap, and mining. The node bootstraps to the network automatically (built-in seeds + peer exchange) and starts mining to your wallet.
  4. Back up your recovery phrase immediately — it is stored only on this machine; if you lose it, your coins are gone for good.

⚠ First-launch security warning (unsigned builds)

These builds are not code-signed, so the OS will warn you the first time. Verify the checksum, then:

macOS — “Obscura is damaged / cannot be opened”

Gatekeeper quarantines unsigned apps. After unzipping, clear the quarantine flag, then open:

xattr -dr com.apple.quarantine /path/to/Obscura.app
open /path/to/Obscura.app

Or: right-click the app → Open → Open in the dialog (first run only).

Windows — “Windows protected your PC” (SmartScreen)

Click More info → Run anyway. If a zone block remains, unblock the file first:

Right-click Obscura.bat → Properties → check "Unblock" → OK
Linux — make it executable 
tar xzf Obscura-linux-amd64.tar.gz
cd Obscura-linux-amd64
chmod +x obscura obscura-node
./obscura

Verify the checksum

# macOS / Linux — compare against SHA256SUMS.txt
shasum -a 256 Obscura-darwin-arm64.zip
# Windows (PowerShell)
Get-FileHash .\Obscura-windows-amd64.zip -Algorithm SHA256

The printed hash must match the line for your file in SHA256SUMS.txt. If it doesn't, do not run the file.

Prefer to build from source?

git clone <repo> && cd obscura
CGO_ENABLED=0 go build -o obscura-node ./cmd/obscura-node
./obscura-node --ui --mine        # wallet + swap + mining desktop app

Pure Go, CGO-free — cross-compiles to every OS. See the quickstart.
